Why On-Chain

No Third-Party Dependencies

Pure RPC calls directly to Solana nodes. No reliance on external data providers or aggregators that could introduce latency or failures.

Real-Time Accuracy

Live data straight from the blockchain, not cached aggregator data. Get the most up-to-date validator performance metrics as they happen.

Full Transparency

Verify every data point yourself. All metrics are derived from verifiable on-chain sources, giving you complete confidence in the data integrity.

API Endpoints

GET/api/validators
Get full list of Solana validators with performance metrics
GET/api/validators/:voteAccount
Detailed information for a specific validator
GET/api/validators/:voteAccount/history
Historical performance data for a validator
GET/api/epoch/current
Current epoch information and statistics
GET/api/stake-accounts/:wallet
Stake account details for a specific wallet
GET/api/network/stats
Network statistics including Nakamoto coefficient
GET/api/validators/compare
Compare multiple validators side by side
GET/api/validators/top
Validator leaderboard by various metrics
GET/api/alerts/delinquent
Alert system for delinquent validators
WebSocket/ws
Real-time updates via WebSocket connection
